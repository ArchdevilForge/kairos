# 仓位管理模板

## 资金分配原则

### 固定仓位法
将总资金等份划分，每次只用固定金额开仓。

**示例**：
- 总资金：30,000 USDT
- 分成3份：每份10,000 USDT
- 每次开仓：10,000 USDT

### 动态调整
- **亏损时**：从场外资金补进，保持每份金额不变
- **盈利时**：把利润提现，不增加单份金额
- **跃升条件**：资金大幅增长（如翻倍）后才提高每份金额

## 杠杆限制

### 杠杆上限
| 品种 | 最大杠杆 | 推荐杠杆 |
|------|----------|----------|
| BTC/USDT | 10x | 3-5x |
| ETH/USDT | 10x | 3-5x |
| 主流山寨 | 5x | 2-3x |
| 小市值山寨 | 3x | 1-2x |
| 套利对冲 | 3x | 2x |

### 杠杆选择原则
1. **盈亏比高**：可以用较高杠杆
2. **止损较远**：必须降低杠杆
3. **市场不确定**：用最低杠杆
4. **新手入门**：一律用最低杠杆

## 仓位计算公式

### 基本公式
```
仓位大小 = (资金 × 风险比例) / (止损距离 / 进场价)
```

### 示例计算
**场景**：
- 资金：10,000 USDT
- 风险比例：33%（每份资金）
- 进场价：68,000
- 止损价：67,200
- 止损距离：800（1.18%）

**计算**：
```
仓位大小 = (10000 × 0.33) / (800 / 68000)
         = 3300 / 0.01176
         = 280,612 USDT
         = 4.13 BTC
```

**杠杆**：
```
杠杆 = 仓位大小 / 资金
     = 280612 / 10000
     = 28x（超过限制）
     
调整后：使用5x杠杆
实际仓位 = 10000 × 5 = 50,000 USDT = 0.735 BTC
```

## 风险控制

### 单笔风险限制
- **最大单笔仓位**：总资金的33%
- **最大单笔亏损**：总资金的3%（止损10%距离）

### 总体风险限制
- **最大总暴露**：总资金的66%
- **最大日亏损**：总资金的10%
- **最大连续亏损**：5次

### 风险检查清单
开仓前必须检查：
- [ ] 单笔仓位 ≤ 33%
- [ ] 总暴露 ≤ 66%
- [ ] 日亏损未超限
- [ ] 连续亏损 < 5次
- [ ] 止损已设置
- [ ] 盈亏比 ≥ 2:1

## 分仓策略

### 3份分仓法
```
总资金: 30,000 USDT
├── 第1份: 10,000 USDT (用于试仓)
├── 第2份: 10,000 USDT (用于加仓)
└── 第3份: 10,000 USDT (用于机会)
```

### 使用时机
- **第1份**：市场春天，开始试仓
- **第2份**：趋势确认，加仓扩大战果
- **第3份**：极端机会，重仓出击

### 加仓条件
1. **底仓盈利**：已有安全垫
2. **趋势明确**：主升浪进行中
3. **结构完美**：二次突破或企稳
4. **盈亏比高**：加仓后整体盈亏比仍佳

## 止损设置

### 止损类型
1. **结构止损**：放在有效防守位
2. **固定止损**：固定百分比（如5%）
3. **移动止损**：随价格移动

### 止损位置
| 策略 | 止损位置 | 说明 |
|------|----------|------|
| 箱体突破 | 箱体下沿 | 跌破下沿=假突破 |
| 箱底承接 | 箱体最下沿 | 跌破最下沿=趋势反转 |
| 双底拐点 | 双底最低点 | 破新低=下跌未结束 |
| 追涨单 | 最近有效支撑 | 防守距离大=仓位轻 |

### 推原价保护
当行情脱离成本区后：
```
新止损位 = max(原止损位, 进场价)
```
确保该笔交易不再亏损。

## 止盈策略

### 分批止盈
不要追求卖在最高点，分批出货。

**示例**：
- 目标1：+5% → 平仓30%
- 目标2：+10% → 平仓30%
- 目标3：+20% → 平仓40%

### 止盈位置
1. **前期高点**：筹码密集区
2. **整数关口**：心理关口
3. **斐波那契**：1.618、2.618等
4. **通道上轨**：上升通道上沿

### 动态止盈
根据市场阶段调整：
- **主升浪初期**：可以格局，看更高
- **主升浪后期**：见好就收，边拉边跑
- **高位震荡**：快速兑现，不贪心

## 套利仓位管理

### 套利仓位限制
- **单笔套利**：总资金的10%
- **总套利暴露**：总资金的30%
- **同时套利对**：最多3对

### 套利杠杆
- **跨所套利**：2-3x
- **费率套利**：2x
- **统计套利**：3x

### 套利止损
- **价差扩大**：止损设置在价差扩大50%处
- **单腿亏损**：单腿亏损超过5%强制平仓
- **时间止损**：持仓超过7天未收敛

## 心态管理

### 仓位与心态
- **轻仓**：心态平和，决策理性
- **重仓**：心态紧张，容易犯错
- **满仓**：心态崩溃，无法执行纪律

### 失控时的应对
1. **立即减仓**：降低到舒适水平
2. **暂停交易**：离开盘面冷静
3. **复盘分析**：找出问题根源
4. **回归系统**：严格执行纪律
